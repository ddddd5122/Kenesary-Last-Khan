# План добавления сцены с Ағыбай батыром

## Контекст
- Сцена вставляется внутри каждой ветки меню, сразу после реплики Кенесары.
- Kенесары смещается влево, Ағыбай появляется справа.
- Масштаб обоих персонажей: 0.7.

## Изменения в `game/script.rpy`
1. Добавить объявление образа:
   - `image agybai_batyr = "images/characters/agybai_batyr.png"`
2. В каждой ветке меню после реплики Кенесары:
   - показать Kенесары слева (`left`) с `zoom 0.7`
   - показать Ағыбай справа (`right`) с `zoom 0.7`
   - (опционально) `with dissolve` или без перехода — оставить как есть, если не нужно
3. Добавить озвучку `voices/ab_1.mp3` тремя репликами с таймингами:
   - `voice "<from 0.0 to 2.981>voices/ab_1.mp3"` → «Хан ие! Шабармандардан хабар келді»
   - `voice "<from 2.981 to 9.503>voices/ab_1.mp3"` → «Ақмола приказы жағынан казактар отряды жылжып келеді. Олар біздің жиынымызды біліп қойған сияқты!»
   - `voice "<from 9.503>voices/ab_1.mp3"` → «Не істейміз? Жауды орман ішінде қарсы аламыз ба? Әлде иен далаға тартып әбден қалжыратып барып соғамыз ба?»
4. `return` оставляется после меню (или переносится ниже, если дальше будет продолжение сцены).

## Итоговая структура ветки (описательно)
- реплика Кенесары
- показать Kенесары слева
- показать Ағыбай справа
- 3 реплики Ағыбая с `voice` сегментами
- дальнейший общий поток
